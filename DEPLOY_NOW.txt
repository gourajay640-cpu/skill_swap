🚀 NEXT: Deploy Backend to Heroku (Optional)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

After frontend is live, deploy backend:

  1. Install Heroku CLI: brew install heroku
  2. Login: heroku login
  3. Create app: heroku create skillswap-api
  4. Set environment variables:
     heroku config:set JWT_SECRET="your-secret"
     heroku config:set MONGODB_URI="your-mongodb-uri"
  5. Deploy: git push heroku main

Your backend will be at: https://skillswap-api.herokuapp.com
